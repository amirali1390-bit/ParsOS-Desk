### About

**ParsOS Desk** is a graphical operating system built entirely with the [Cosmos](https://github.com/CosmosOS/Cosmos) toolkit and **C#**. It provides a full desktop-like experience directly on bare metal, including a window manager, custom rendering pipeline, and multimedia decoding.

### ✨ Features

- 🪟 **Window Manager** — draggable, resizable, and independently titled windows.
- 🔤 **TrueType Font (TTF) Renderer** — with BiDi and Gray Scale AA.
- 🎵 **MP3 Decoder** — built-in audio decoding and playback support.(in development)
- 🖼️ **PNG Decoder** — native image decoding for pictures and wallpapers.
- 🎬 **Animations** — smooth UI transitions.
- 💠 **Blur Effects** — Blur visual effects for lock screen.
- 🖱️ **Full Window Interactions** — move and resize and manage windows.

### 🧱 Built With

- [Cosmos OS](https://github.com/CosmosOS/Cosmos) — Open-source, managed operating system toolkit
- **C#** / .NET — core development language
- Custom-built renderer, window manager, and media decoders

## فارسی

> برو به: [English ⬆️](#-english)

### درباره‌ی پروژه

**ParsOS Desk** یک سیستم‌عامل گرافیکی است که به‌طور کامل با استفاده از فریم‌ورک [Cosmos](https://github.com/CosmosOS/Cosmos) و زبان **#C** ساخته شده است. این سیستم‌عامل یک محیط دسکتاپ کامل را مستقیماً روی سخت‌افزار (Bare Metal) فراهم می‌کند؛ شامل مدیریت پنجره‌ها، موتور رندر اختصاصی و دیکود چندرسانه‌ای — همگی با کد مدیریت‌شده (Managed Code).

### ✨ امکانات

- 🪟 **مدیریت پنجره‌ها (Window Manager)** — قابلیت جابه‌جایی، تغییر اندازه و نام‌گذاری پنجره‌ها
- 🔤 **رندرکننده‌ی فونت TTF** — موتور اختصاصی رندر متن برای نمایش فونت‌های شفاف و مقیاس‌پذیر
- 🎵 **دیکودر MP3** — پشتیبانی داخلی از دیکود و پخش فایل‌های صوتی
- 🖼️ **دیکودر PNG** — دیکود بومی تصاویر برای آیکون‌ها، والپیپرها و رابط کاربری
- 🎬 **انیمیشن‌ها** — افکت‌های حرکتی و ترانزیشن‌های نرم در رابط کاربری
- 💠 **افکت بلور (Blur)** — جلوه‌ی بصری بلور و شیشه‌ی مات به‌صورت آنی برای پنجره‌ها و اجزای رابط کاربری
- 🖱️ **تعامل کامل با پنجره‌ها** — جابه‌جایی و تغییر اندازه پنچره ها.

### 🧱 ساخته‌شده با

- [Cosmos OS](https://github.com/CosmosOS/Cosmos) — فریم‌ورک متن‌باز برای ساخت سیستم‌عامل مدیریت‌شده
- **#C** / دات‌نت — زبان اصلی توسعه
- موتور رندر، مدیریت پنجره و دیکودرهای رسانه‌ای، همگی اختصاصی و دست‌ساز

### 🚀 شروع به کار

**پیش‌نیازها:**
- ویژوال استودیو ورژن 2022 یا 2026.
- نصب بودن Cosmos User Kit
- نرم‌افزار QEMU / VirtualBox / VMware برای تست، یا یک فلش بوت‌شونده برای اجرا روی سخت‌افزار واقعی
